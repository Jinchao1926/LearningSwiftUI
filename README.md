# SwiftUI-Learning
Some demos while learning SwiftUI, each directory is an independent demo project.

- [Calculator](./Calculator): A simple calculator app.
- [PokeMaster](./PokeMaster): A Pokémon manager app for browsing and managing Pokémon.
- [Chess](./Chess): A tic-tac-toe game.
- [Faker](./Faker): A demo app simulating the Joy City Beijing shopping mall experience.
- [SwiftDemos](./SwiftDemos): A collection of demos showcasing various Swift and SwiftUI features.
